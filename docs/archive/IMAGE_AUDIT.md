# Azur Menton Apartment Image Audit

Generated from files in `public/images/apartments/`. No images were deleted, replaced, upscaled or AI-enhanced.

## Summary

- Most apartment photos are launch-ready only when displayed in constrained editorial frames.
- Many portrait/detail images are roughly 500-760 px wide/high and should not be used as full-width hero images.
- Detail page hero layouts should avoid full-bleed backgrounds and use split/collage compositions with max rendered widths around 420-760 px depending on the source.
- Bathroom, kitchen, parking, entrance and location-only photos should remain lower-priority gallery/supporting images.
